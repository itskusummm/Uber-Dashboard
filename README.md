# 🚖 Uber Data Analysis Dashboard (Power BI)

## 📌 Project Overview

This project is an end-to-end **data analysis and visualization dashboard** built using **Power BI**, focused on uncovering key insights from Uber ride data.

The dashboard provides a clear, interactive view of:

* Booking trends 📈
* Revenue performance 💰
* Customer & driver behavior ⭐
* Payment preferences 💳
* Operational efficiency 🚗

The goal of this project is to simulate **real-world business decision-making** using data.

---

## 🎯 Key Insights

* **62K total completed bookings** with noticeable monthly fluctuations
* **Revenue peaked consistently around 3M+**, showing stable demand
* **Auto generated the highest revenue**, followed by Bike and Go Mini
* **UPI dominates payment methods**, indicating strong digital adoption
* **Top pickup location:** Khandsa
* **Top drop location:** Adarsh Nagar
* Customer rating (**4.43**) slightly higher than driver rating (**4.23**) → potential improvement area

---

## 📊 Dashboard Features

* 📅 **Time-based analysis** (Monthly trends)
* 🚘 **Vehicle-wise revenue comparison**
* 💳 **Payment method distribution**
* 📍 **Location-based insights**
* ⭐ **Customer & driver rating analysis**
* ❌ **Lost vs Completed booking tracking**
